Project Description

This project is a fully self-designed robotic arm developed for the 2024 Engineering Symposium. All components were created from scratch, 3D modeled in Fusion 360, and manufactured using 3D printing. The system is controlled by an Arduino Uno with integrated Bluetooth functionality, allowing wireless operation via smartphone. The project reflects a combination of mechanical design, electronics, and embedded systems skills.

Technical Specifications

  • Design: Fully custom 4-axis robotic arm with modular architecture
  • CAD Software: Autodesk Fusion 360 for complete mechanical design
  • Manufacturing: 3D printed using PETG material on Bambu Lab X1 Carbon
  • Control System: Arduino Uno microcontroller with custom C++ programming
  • Connectivity: Bluetooth module for wireless smartphone control
  • Actuators: Servo motors for precise joint control
