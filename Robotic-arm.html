Project Description

This project is a fully self-designed robotic arm developed for the 2024 Engineering Symposium. All components were created from scratch, 3D modeled in Fusion 360, and manufactured using 3D printing. The system is controlled by an Arduino Uno with integrated Bluetooth functionality, allowing wireless operation via smartphone. The project reflects a combination of mechanical design, electronics, and embedded systems skills.

Key Features

  • Fully custom-designed 4-axis robotic arm developed for the 2024 Engineering Symposium
  • All mechanical components designed from scratch in Fusion 360
  • 3D printed parts using PETG material on the Bambu Lab X1 Carbon printer
  • Arduino Uno-based control system with Bluetooth wireless operation via smartphone
  • Modular design for easy upgrades and experimentation
  • Designed and assembled both mechanical and electrical systems independently

Challenges

  • Originally designed as a 6-axis robotic arm, but motor torque limitations required simplifying to a 4-axis system
  • Built entirely from scratch outside of academic modules, requiring independent learning and initiative
  • Managed the project alongside full-time studies, society involvement, and a part-time job at LANA Asian Street Food
  • Faced challenges with hardware limitations, time constraints, and resource balancing
  • Motivated by personal interest and a desire to demonstrate both technical and soft skills
