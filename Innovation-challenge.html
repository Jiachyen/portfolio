Project Description

As part of an Innovation Challenge, our team developed a user-centered assistive device designed to improve the daily lives of individuals with physical limitations. The project involved extensive research through site visits, interviews with occupational therapists, and direct consultations with end-users and their families. Using insights from this research, we created a functional prototype—3D-printed with sustainable materials selected via Granta EduPack—to ensure environmental responsibility in line with the UN Sustainable Development Goals. The project was presented at a conference and proudly recognized as the best solution.

Key Features

  • Human-centered design informed by interviews and real-world feedback
  • Prototyped using eco-friendly materials for sustainability and accessibility
  • Functional 3D-printed model tailored to user needs
  • Collaborative workflow using digital tools for remote teamwork
  • Final solution selected as the best solution at the conference
