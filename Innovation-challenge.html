Project Description

As part of the MTU Innovation Challenge 2024, our team developed a user-centered assistive device designed to improve the daily lives of individuals with physical limitations. The project involved extensive research through site visits, interviews with occupational therapists, and direct consultations with end-users and their families.

Using insights from this comprehensive research, we created a functional prototype—3D-printed with sustainable materials selected via Granta EduPack—to ensure environmental responsibility in line with the UN Sustainable Development Goals. The project was presented at a conference and proudly recognized as the best solution.

Design Process & Methodology

  • User Research: Conducted site visits and interviews with occupational therapists
  • Stakeholder Engagement: Direct consultations with end-users and their families
  • Human-Centered Design: Iterative design process based on real user feedback
  • Sustainability Focus: Material selection using Granta EduPack for environmental responsibility
  • Prototype Development: Functional 3D-printed model tailored to specific user needs
  • Collaborative Workflow: Remote teamwork using digital collaboration tools

Key Features & Innovations

  • Accessibility-First Design: Prioritized user needs and accessibility requirements
  • Sustainable Materials: Eco-friendly material selection aligned with UN SDGs
  • Functional Prototype: Working 3D-printed model demonstrating practical application
    • Evidence-Based Design: Design decisions informed by professional consultation
  • Scalable Solution: Framework can be adapted for various assistive technology applications
  • Professional Recognition: Awarded best solution at the innovation challenge

Technical Implementation

  • CAD Design: Comprehensive 3D modeling using Fusion 360
  • Additive Manufacturing: 3D printing for rapid prototyping and testing
  • Material Analysis: Granta EduPack for sustainable material selection
  • Digital Collaboration: Microsoft Teams and Google Drive for remote teamwork
  • User Testing: Iterative testing with target user groups
